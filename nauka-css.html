Klasy CSS

Klasy w CSS Dotychczas poznałem stylowanie po elementach.
Można to robić w pliku html lub stowrzyć odrębny plik css w którym stylujemy nasz główny dokument.
Oprócz stylowania po elementach, występuje jeszcze kilka form stylowania, z czego najważniejszą i najbardziej powszechną jest forma stylowania po .klasach.
Stylowanie po klasach:
Nadajemy danemu elementowi klasę poprzez atrybut class=”nazwa-atrybutu”
W pliku css będziemy tworzyć regułę “.nazwa-atrybutu {}”.
Można również stylować po id. W tym celu nadajemy elementowi atrybut id=”nazwa”.
W css tworzymy regułę “#nazwa {}”.
W przypadku id należy pamiętać, że id nadajemy jedno, i dotyczy ono jednego elementu.
Oprócz tego jest jeszcze stylowanie INLINE-owe bezpośrednio w pliku głownym przy danym elemencie.
Staramy się tego unikać, używać w wyjątkowych sytuacjach. Tak samo jak ostatniego rodzaju stylowania czyli !important, który nadpisuje wszelkie inne style.
Jest jak złoty metal na olimpiadzie. important wpisujemy w miejscu, w którym dana właściwość ma być na wierzchu.
Istnieje jeszcze coś takiego jak łączenie selektorów.
Możemy za pomocą tego ostylować tylko daną część elementów, która w drzewie zagnieżdżeń występuje jako dziedzic innego konkretnego znacznika.
Specyficzność: ! ( _ ), InLINE (_), #name _, .class_, znacznik_
